# toyBloch
 A small Bloch simulator

# Environment Setup
(1) conda create -n bloch_env python=3.12.12
(2) pip install -r requirements.txt
(3) pip install scipy
(4) Go to "examples-->hyperbolic_secant.ipynb", create an extra cell at the top:
    import sys
    sys.path.append("path to toybloch folder")
